# 팀원 서비스 폴더

이 폴더에 새로운 서비스를 추가하세요.

## 구조 예시

```
teammate/
├── index.html          (서비스 메인 페이지)
├── css/                (스타일시트)
├── js/                 (스크립트)
└── ...
```

## 가이드

1. 이 폴더 이름을 본인 이름으로 변경하세요 (예: `teammate` → `jane`)
2. `index.html`을 서비스 메인으로 사용하세요
3. 루트 `index.html` (포털 페이지)에 서비스 카드를 추가하세요
4. CSS는 Tailwind CDN 또는 자체 스타일시트 사용 가능
